# Notion 설정

이 파일은 notion-summary 스킬에서 사용하는 Notion 연동 설정입니다.

## 기본 페이지

업로드 대상 Notion 페이지 설정:

- **페이지 ID**: 2a83077f85aa806a8e75f920a990b589
- **페이지 이름**: Claude 작업 기록

> 페이지 ID 확인 방법:
> 1. Notion 페이지 열기
> 2. "Share" 버튼 → "Copy link"
> 3. URL에서 페이지 ID 추출: `notion.so/페이지이름-{페이지ID}`
> 4. 페이지 ID는 32자리 16진수 (하이픈 포함 36자리)

## 업로드 설정

- **대상 타입**: database
- **날짜별 하위 페이지**: true
- **프로젝트별 분류**: true
- **기본 프로젝트명**: general

> 대상 타입 옵션:
> - `database`: Notion 데이터베이스에 항목 추가 (권장)
> - `page`: 일반 페이지 하위에 새 페이지 생성
